## ABOUT

Mug print constructor that allows user to configure images, texts and apply it onto 3D model.

  

## FEATURES

 - Ability to upload your pictures (up to 20)
 - Ability to transform them (loc rot scale)
 - Easy layer system
 - Text image generator
 - 3D visualization
